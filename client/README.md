# 3DP-Manager — Frontend (React + MUI)

Admin panel UI for 3DP-Manager. Built with React 19, TypeScript, Vite, and Material UI.

## Setup

```bash
npm install
```

## Development

```bash
# start dev server with HMR
npm run dev

# lint
npm run lint

# type checking
npm run typecheck
```

## Test

```bash
# run tests
npm run test

# watch mode
npm run test:watch

# coverage
npm run test:cov
```

## Build

```bash
npm run build
```

## Tech Stack

- **React 19** — UI framework
- **TypeScript** — type safety
- **Vite** — build tool
- **Material UI 7** — component library
- **@mui/x-data-grid** — data tables with sorting/filtering/pagination
- **React Router 7** — routing
- **Axios** — HTTP client
- **Vitest** — test runner
